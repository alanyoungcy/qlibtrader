## 🚀 **Quick Start**
```bash
# Clone the repository
git clone https://github.com/alanyoungcy/qlibtrader.git
cd qlibtrader

# Run setup script
./setup.sh

# Activate virtual environment
source venv/bin/activate

# Start the trading system
python working_trading_app.py
```
